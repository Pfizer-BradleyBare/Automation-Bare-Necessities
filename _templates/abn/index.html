Sounds Like a Lotta Hoopla Over Automation... What is ABN?

Low Code Automation

ABN methods are built using color coded blocks in a consecutive format. These blocks provide the full functionality of your automation system without the low level minutae. Additionally, you can create psuedo parallel processes within a single method file to enable complex DOE work.

System Agnostic

ABN is fully independant of your automation system. Methods are built using a low code excel based editor then translated to your automation system on the fly. This allows you, the analyst, to focus on science while ABN focuses on robust automation.

Full Service

ABN will completely handle all aspects of your automated method. Deck loading, error handling, preheating / precooling, pipette selection, and more. However, if desired you will still have complete control over all aspects of your method.

Method Multithreading

ABN has the possibility to schedule methods and orchastrate resources of larger automation systems. You can define your desired completion time and then ABN will complete methods around your schedule, not the other way around.

Worklist Interface

ABN Blocks can interface directly to a sample worklist, allowing blocks to treat samples uniquely during execution. This is a simpler option to parallel processing for DOE workflows on automated systems.

Enhanced Notifications

ABN notifications enhance what is normally possible on automated systems. Emaila and text notifications are available with response support. Tell the notification how you would like to proceed then ABN will continue as instructed.
